# Changelog — Professional AI Governance Framework (Skill)

All notable changes to this Skill are documented here.

## [1.0.0] — 2026-07-12

First public release of the generalized Skill.

### Added
- Founding principle and non-negotiable rules, phrased in a multi-sector-neutral way.
- G0 normative framing mechanism: the AI questions the professional about the applicable reference framework instead of asserting it; any unconfirmed suggestion remains explicitly flagged "to be verified."
- Validation gates G0 through G5, with generalized passage criteria and decision-maker.
- Criticality classification N1 through N4, with compared examples for engineering, medical, legal, and finance/accounting.
- Source registry template.
- Sectoral application examples (engineering detailed; law, medicine, finance sketched).
- Dual licensing regime: general framework under CC BY-ND 4.0, SYVEX application annexes remaining proprietary.

### Origin
This Skill generalizes the *SYVEX AI Framework v1.2.1* (internal use, SYVEX ENGINEERING, unpublished), developed and refined in production since 2025 on geotechnical and structural engineering projects.
